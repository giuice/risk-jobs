State variables:

```javascript
let selectedOccupation = null;

```

`renderCards()` — Iterates `OCCUPATIONS` array. For each occupation, creates a `div.card` with:
- `data-id` attribute set to occupation.id
- Inner HTML: `<div class="card-name">{name}</div><div class="card-risk">Risk: {baseScore}/10</div>`
- Click handler calls `selectCard(id)`
- Appends each card to `#card-grid`

`renderExperienceSelector()` — Iterates `SENIORITY_LEVELS` array. For each level, creates a `button.exp-btn` with:
- `data-id` attribute set to level.id
- Inner HTML: `<span>{label}</span><span class="exp-label">{range}</span>`
- Click handler calls `selectExperience(id)`
- Appends each button to `#experience-selector`

`selectCard(id)` — Sets `selectedOccupation = id`, removes `.selected` from all `.card` elements, adds `.selected` to the clicked card, calls `updateCTA()`

`selectExperience(id)` — Sets `selectedSeniority = id`, removes `.selected` from all `.exp-btn` elements, adds `.selected` to the clicked button, calls `updateCTA()`

`updateCTA()` — If both `selectedOccupation` and `selectedSeniority` are not null, remove `disabled` attribute from `#cta-btn`. Otherwise, set `disabled`.

`showResults()` — Sets `#input-view` display to `none`, sets `#results-view` display to `block`

`showInput()` — Sets `#results-view` display to `none`, sets `#input-view` display to `block`

**BOOT section — fill between `===== BOOT =====` and end of script:**
```javascript
function init() {
  renderCards();
  renderExperienceSelector();
  document.getElementById('cta-btn').addEventListener('click', function() {
    if (selectedOccupation && selectedSeniority) {
      showResults();
    }
  });
  document.getElementById('try-again-btn').addEventListener('click', function() {
    showInput();
  });
}

document.addEventListener('DOMContentLoaded', init);
```

NOTE: All DOM access is in the UI and BOOT sections ONLY. The ENGINE section between its markers must remain pure functions with zero DOM access.
